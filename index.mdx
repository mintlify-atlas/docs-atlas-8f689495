The Dashlane Public API provides secure, read-only programmatic access to your Dashlane team account. Use it to retrieve member data, monitor device activity, analyze password health trends, and build custom integrations with your existing enterprise systems — all authenticated with a simple Bearer token.

## Get started in three steps

<Steps>
  <Step title="Generate an API key">
    Create an API key from the [Dashlane Admin Console](https://support.dashlane.com/hc/en-us/articles/23955544757266-Dashlane-public-API) or using the Dashlane CLI:
    ```bash
    dcli team public-api create-key "My integration key"
    ```
    Your key will be formatted as `Bearer DLP_teamUuid_accessKey_secretKey`. Store it securely — it is shown only once.
  </Step>
  <Step title="Authenticate your request">
    Pass the API key in the `Authorization` header of every request:
    ```bash
    curl -X POST 'https://api.dashlane.com/public/teams/Members' \
      -H 'Content-Type: application/json' \
      -H 'Authorization: Bearer DLP_teamUuid_accessKey_secretKey' \
      -d '{}'
    ```
  </Step>
  <Step title="Explore the API">
    Browse the [API Reference](/api-reference/overview) or download the [OpenAPI 3.0 specification](https://get.dashlane.com/public-api/openapi.json) to integrate with your tooling.
  </Step>
</Steps>

<Note>
  The Dashlane Public API is **read-only**. All endpoints use the `POST` method with a JSON body, and return JSON responses. No data is modified by any API call.
</Note>
